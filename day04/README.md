## How to run locally
Type for example:
```bash
part=part1 dotnet run
part=part2 dotnet run
```

## How to build and run with Docker
Type for example:
```bash
docker build -t day04 .
docker run --rm -e part=part1 day04
docker run --rm -e part=part2 day04
```
